# A01 — Telecommunications Fundamentals

APA-style paper answering five foundational questions about telecommunication systems (week of June 8, 2026).

**What I did.** Defined a telecommunications system and its modern scope; explained the transmitter → channel → receiver model with a practical BPSK example; contrasted analog and digital communication and the metrics used to judge them (bandwidth, SNR, latency, throughput, jitter, error rate); described the four communication types (voice, data, video, multimedia) and how topology and architecture shape their transmission; and compared star, mesh, ring, and bus topologies on performance, reliability, and scalability.

**What I learned.** Every network conversation reduces to the same three roles — something that prepares the message, a path that inevitably degrades it, and something that recovers it. Digital won because bits can be encoded, checked, and regenerated, so noise doesn't accumulate. Topology is a set of trade-offs, not a ranking: star is practical, mesh is resilient, ring is predictable, bus is cheap but crowded.
